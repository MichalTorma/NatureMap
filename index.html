      Welcome to NatureMap

      The Data Source

      You're viewing live biodiversity data from GBIF — the Global Biodiversity Information Facility. This open-access network provides anyone, anywhere, with data about all types of life on Earth.

      Explore the Features

      • High Precision

        Zoom in (level 12+) to see individual vector markers with exact coordinates and timestamps.

      • Flexible Layers

        Switch base maps, toggle conservation overlays, and customize biodiversity heatmap styles.

      • Deep Context

        Tap any marker to explore species taxonomies, vernacular names, and live Wikipedia previews.
